public-configs
==============

Some configs from ~/ for all 

Config for use
--------------

In ~/.bashrc or ~/.bash_aliases

```shell
alias = '~/where/you/clone/this/repo/tmux.sh'
```

In ~/.tmux.conf

```shell
source-file /home/eugene/Projects/shell/public-configs/tmux.conf
```

Then

```shell
$ source ~/.bashrc
```

or 

```shell
$ source ~/.bash_aliases
```
